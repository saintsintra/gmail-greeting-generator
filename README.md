Privacy Policy
Gmail Greeting Generator
Effective Date: March 11, 2026
Last Updated: March 11, 2026

Overview
Gmail Greeting Generator ("the Extension") is a Chrome browser extension that helps users insert randomized, friendly greetings into Gmail compose windows. This Privacy Policy explains what data the Extension does and does not collect, and how it operates.
The short version: we collect nothing. The Extension operates entirely within your browser and does not transmit any data to any server.

Data We Do Not Collect
Gmail Greeting Generator does not collect, store, transmit, or share any personal data of any kind. Specifically, the Extension does not collect:

•	Your name, email address, or any personal identifiers
•	The contents of your emails, drafts, or compose windows
•	Your Gmail contacts, inbox, or any email metadata
•	Browsing history or activity on Gmail or any other website
•	Device information, IP addresses, or location data
•	Usage statistics, analytics, or behavioral data
•	Any information that could be used to identify you

All greeting generation happens locally within your browser using JavaScript. No data leaves your device.

Permissions Explained
The Extension requests the following Chrome permissions, each of which is used solely to enable core functionality:

activeTab
Allows the Extension to interact with the Gmail tab you are currently viewing. This is used only to detect the Gmail compose window and insert a greeting into it. The Extension cannot access any other tabs.

scripting
Allows the Extension to run its greeting-insertion script within the Gmail page. This script only reads and writes to the email compose area — it does not read your inbox, sent mail, or any other part of Gmail.

clipboardWrite
Allows the Extension to copy a generated greeting to your clipboard when you click "Generate" in the popup. This is a one-way write operation only — the Extension cannot read the contents of your clipboard.

Host Permission: mail.google.com
Restricts the Extension's activity exclusively to Gmail. It cannot operate on any other website.

Third-Party Services
Gmail Greeting Generator does not use any third-party services, analytics platforms, advertising networks, or external APIs. There are no trackers, SDKs, or remote connections of any kind embedded in the Extension.

Local Storage
The Extension does not use browser storage (localStorage, sessionStorage, or chrome.storage) to retain any information between sessions. Each time you open the popup, a greeting is freshly generated in memory and discarded when the popup closes.

Children's Privacy
The Extension is not directed at children under the age of 13 and does not knowingly collect any information from children. Because the Extension collects no data from any users, this concern does not apply in practice.

Changes to This Policy
If the Extension is updated in a way that changes its data practices, this Privacy Policy will be updated accordingly, and the "Last Updated" date at the top of this document will reflect the change. Continued use of the Extension after any update constitutes acceptance of the revised policy.

Contact
If you have any questions about this Privacy Policy or the Extension's data practices, please open an issue in the GitHub repository or contact the developer directly through the Chrome Web Store listing.
